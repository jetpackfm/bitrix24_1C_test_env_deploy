# Развёртывание тестового стенда: Bitrix24 + Интернет-магазин + интеграция с 1С

## Цель проекта
Развернуть тестовый стенд с Bitrix24 (CRM + Интернет-магазин) и настроить интеграцию с 1С:Предприятие (Управление торговлей) с использованием учебных версий программного обеспечения.

## Ограничения учебной версии 1С

Перед началом работы следует учитывать ограничения учебной версии платформы 1С:Предприятие:

- Работает только в файловом режиме (клиент-серверная архитектура не поддерживается)
- Не позволяет полноценно обрабатывать документы (заказы) в рамках обмена с Битрикс24
- Для полной функциональности требуется обновление конфигурации до актуальной версии (требуется подписка ИТС)

### Что работает
- ✅ Выгрузка товаров, цен и остатков из 1С на сайт
- ✅ Синхронизация справочников (единицы измерения, характеристики)
- ✅ Обновление статусов заказов на сайте

### Что не работает (в учебной версии)
- ❌ Полноценная обработка заказов в 1С (ошибка клиент-серверного режима)
- ❌ Автоматическое создание документов на основании заказов

Для промышленного использования необходима полноценная лицензионная версия 1С:Предприятие и подписка ИТС.
