# 4. Создание интернет-магазина

## Создание нового сайта

1. В админке: **Настройки → Настройки продукта → Сайты → Список сайтов**
2. Нажать **«Добавить сайт»**
3. Заполнить поля:
   - **ID:** `s2` (s1 занят основным порталом)
   - **Название:** произвольное (например, `eshop`)
   - **Папка сайта:** `/newshop/` (не использовать корень `/` и папку `/bitrix` — они заняты ядром)
4. Нажать **«Сохранить»**

## Установка магазина

После сохранения откроется мастер установки 1С-Битрикс: Управление сайтом.

1. Выбрать решение: **Интернет-магазин**
2. Для тестового стенда принять все настройки по умолчанию (нажимать «Далее»)
3. Дождаться завершения установки

## Проверка работоспособности

1. Перейти на сайт магазина: `http://b24.test/newshop/`
2. Открыть любой каталог, выбрать товар
3. Добавить товар в корзину и удалить — функционал должен работать корректно
